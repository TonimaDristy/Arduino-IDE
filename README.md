Built using **Arduino IDE** as part of our Human-Computer Interaction (HCI) course, implementing a **Tangible User Interface (TUI)** for a smart, contactless billing experience.

---

## 🧠 About the Project
This system replaces traditional billing methods with a **real-time tangible interaction system** where users simply place a product on a platform and get instant price + identification without manual input.

---

## ⚙️ How It Works

1️⃣ Place the product on the load cell  
2️⃣ Load cell measures the weight  
3️⃣ HX711 amplifier processes the signal  
4️⃣ ESP32-CAM captures the product image  
5️⃣ Image is sent to the server  
6️⃣ Edge Impulse model detects the product  
7️⃣ System calculates total price  
8️⃣ Weight and price are displayed instantly  

---

## 🔄 System Flow
Product → Load Cell → HX711 → ESP32-CAM → Server → AI Model → Price Calculation → Display

---

## ✨ Key Features
- 🎯 Real-time product detection  
- ⚖️ Accurate weight measurement using load cell  
- 📷 AI-based image recognition (Edge Impulse)  
- ⚡ Fast automated billing system  
- 🖥️ No manual input required (TUI-based interaction)
